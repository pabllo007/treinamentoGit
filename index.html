Git

Treinamento de git através de video-aula do alura.

Comandos

  • mkdir - Cria pasta
  • git init - Para iniciar o acesso ao git
  • git ls-files - lista os arquivos que estão sendo gerenciados pelo git
  • git status - mostra o status dos arquivos
  • git add "arquivo" - adiciona o arquivo que será gerenciado pelo git
  • git commit -m "Mensagem para o log" - commit das alteraçõs
  • git config user.name "Nome de qualquer pessoa" - Configura o nome do usuário que está realziando as alterações (Local)
  • git config user.name "email" - configura o email do usuário que está realizando as alterações (Local)
  • git log - lista as alterações realizadas
  • git remote - lista os repositorios configurados
  • git remote add 'nome do repositorio (melhor manter o mesmo nome do projeto que está no servidor)" "url do servidor" - comando para sincronização de repositórios com o servidor
  • git push "nome do repositorio" master (branch principal) - sincroniza as alterações com o servidor
  • git clone "url do repositorio/servidor" - pega o projeto do repositorio/servidor
  • git branch
  • git branch "nome da branch" - Cria uma nova branch
  • git checkout "nome da] branch" - acessa a branch para realizar as alterações
